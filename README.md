# Clock-In System

Agent attendance tracking system for Execo Group.

## Features
- Agent clock in/out interface
- Real-time attendance dashboard
- Role-based access (admin, manager, agent)
- Automated hour calculation
- Export attendance reports

## Tech Stack
- Next.js
- Supabase
- Tailwind CSS
- Vercel (deployment)

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`
